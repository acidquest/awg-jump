# I18n Format

UI в `gateway/frontend` использует словари-модули. Английский словарь является дефолтным, русский добавлен как пример.

## Формат

Каждый словарь — обычный `ts`-модуль с плоским объектом:

```ts
export const en = {
  dashboard: 'Dashboard',
  nodes: 'Entry Nodes',
}
```

## Правила

- ключи должны совпадать между словарями;
- новые страницы добавляют строки в `src/locales/en.ts` и затем в дополнительные языки;
- если перевод отсутствует, UI использует английское значение;
- язык хранится локально и может быть синхронизирован с backend settings.
